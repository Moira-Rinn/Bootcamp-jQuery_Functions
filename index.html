Introduction to jQuery Functions

The quick way to add functionality to your website!

The .click Function

The .click function listens for a click even and then executes a function. In the example below, clicking on the buttons is the click even, and the function calls upon the jQuery functions .hide and .show, respectively. In this instance, this paragraph and the one below will be hidden or shown by targeting the class 'hideShow' which has been assigned to both.

Cick the buttons below to hide and show these paragraphs.

The .hide, .show, & .toggle Functions

The .hide, .show & .toggle functions, can simply hide anything visible, show anything hidden, or toggle between the two. This works by targeting the HTML element, class, or id of whatever you want to hide.

In this example, these functions have an extra attribute assigned to them, duration, which provides a transition effect, rather than the, "now you see it, now you don't" suddeness of the default above.

The . slideUp, .slideDown, & .slideToggle Functions

The .slideUp, .slideDown, and .slideToggle are other transition functions that can be paired with different events to achieve different effects. In this instance they are just being used to hide and show, however they can be paired with hover and scrolling events to make certain features appear to accent the user experience.

The .fadeIn and .fadeOut Function

The .fadeIn and .fadeOut are, as might be obvious, fading transitions. Again they add more graceful animation to different aspects of the site which gives it a more professional feel and nice flow. The speed at which things fade, (as with all transition functions) can be regulated to suit the taste of the space.

The .addClass & .removeClass Functions

The .addClass and .remoeClass functions are a great utility to add and change styling. They allow predefined classes to be added to or remomved from elements, in this case with a click event, but really with any eventlistener. These options present a lot of functionality and utility to really make the site dynamic and interactive.

The .before and .after Functions

The .before and .after functions do pretty much what you would expect, they add somethting either before or after the target element. This is useful in certain situations, for example form validation when there is a error. It can also make the page more interactive and engaging.

The .append Function

The .append Function appends something to the element. It will continue to append that something so long as the eventlistener is triggered. In this instance the click even will continue to add a new sentence to this paragraph as many times as it is clicked.

The .html Function

The .html function adds to or alters the elements HTML. Currently you are reading a paragraph element, but by clicking the button below it will change the element into an unordered list. Many other modifications can be made to transmute the page, making it very versital.

The .attr Function

The .attr, or attribute function modifiies or replaces an attribut of the HTML element. In this instance it adds an ID with a predefined style, but it can be used to change any attribute, which provides a lot of utility in a lot of different instances.

The .val Function

The .val function captures the value of an input element allowing it to be sent or displayed on the page. Here you can choose your favoite color and the button will store it in a variable and print it out below the select box.

The .text Function

The .text function allows jQuery to modify the text of the element. This is very helpful to communicate with users to confirm or rebuke their input. It also alows for a more engaging experience by allowing the user to interact with the site via eventlisteners, such as the click even from pressing the button, and having the site respond.
