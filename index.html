The .click Function

From the documentation:

"This method is a shortcut for .on( "click", handler ) in the first two variations, and .trigger( "click" ) in the third. The click event is sent to an element when the mouse pointer is over the element, and the mouse button is pressed and released. Any HTML element can receive this event.

The .click function is a common function used to call many of the functions below. For example, clicking the buttons below will hide and show this paragraph. Try it!!

The .hide Function

The .hide function, as deminstrated above, can simply hide any, or all HTML elements on the page. However, it also possesses other options that can make the process a bit more graceful.

The .show Function

The .show function, is the opposite of the .hide function and can be used to reveal hidden elements. Like hide, it also possesses other options that can make the process a bit more graceful.
